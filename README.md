netem-webui
===========

A web interface for configuring netem parameters

### Dependencies:
	python-flask
        ([Flask-Socketio] https://flask-socketio.readthedocs.org/en/latest/)

### Start the web app:
	sudo python main.py

### Connect to the web app:
	http://localhost:8080/
